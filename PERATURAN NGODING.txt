PENAMAAN
    Nama Kelas --> menggunakan PascalCase (setiap kata baru menggunakan huruf kapital [UPPERCASE] )
        contoh : IniNamaKelas 
    
    Atribut --> menggunakan snake_case (kata sambung pakai simbol underscore)
        contoh : ini_nama_atribut
    
    Method --> menggunakan camelCase (diawali dengan huruf kecil [lowercase] kemudian jika ada kata 
               baru dituliskan dengan menggunakan huruf kapital [UPPERCASE] )
        contoh : iniNamaMethod
